Building Skills Through Real-World Experience

My programming journey has taken me through internships, education, and personal projects. Each experience taught me something different whether it was working with hardware-integrated software, building cross-platform apps, or developing game systems. These opportunities shaped how I approach problem-solving and software development today.

Key Experiences

  • Internship at Epesi: Worked with FlutterFlow and Dart to build mobile a mobile app to track football game events, learning rapid prototyping and cross-platform development.
  • Internship at Moduvision: Developed software in C# that interacted with autosamplers, gaining hands-on experience with hardware integration and technical debugging.
  • MBO Education in Software Development: Built foundational projects like Pasimo, QR-Goats, and Slidehub using frameworks like Struts 2, Hibernate, and Bootstrap, working in team environments with Scrum methodology.
  • Game Development with Roblox Studio: Created multiple games and systems using Luau, including admin panels, exploration games, and competitive multiplayer minigames, learning event-driven programming and DataStores.

Languages & Technologies

  • Languages: Luau, C#, JavaScript, Java, Dart partially, SQL, HTML/CSS, PHP.
  • Frameworks & Tools: FlutterFlow, Struts 2, Hibernate, Bootstrap.
  • Development Tools: Roblox Studio, Eclipse IDE, VSCode, MySQL Workbench, InteliJ
  • Methodologies: Scrum, version control, debugging, and testing in both solo and team environments

From working on cross-platform mobile apps to building hardware-integrated software and creating multiplayer games, these experiences gave me a diverse skill set. I learned to adapt to different languages and environments, collaborate with teams, and solve complex technical challenges. This foundation continues to drive my growth as a developer.
